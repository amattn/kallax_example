# kallax_example


A test project for the kallax ORM/code generator.

https://github.com/src-d/go-kallax

### Usage

Currently, this project looks for environment variables to connect to a DB.  See the `dbURL()` function in `db_utils.go`.

- DBUSER
- DBPASS
- DBHOST
- DBPORT
- DBNAME
